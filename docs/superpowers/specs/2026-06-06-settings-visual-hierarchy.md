# Settings 页面视觉层级优化

**日期**: 2026-06-06  
**范围**: `frontend/app/dashboard/settings/page.tsx`

## 问题

当前页面有一级 tab（AI 配置 / Tab 2 / Tab 3）+ h2 标题「AI 配置」+ 二级 pill tab（全部/文字模型/生图模型），存在：
- 一级 tab 名称与 h2 标题重复
- h2 标题块隔断了一级 tab 与二级 pill 的视觉父子关系
- 按钮放在右侧，操作距离远

## 方案

去掉 h2 标题区，二级 pill 与「添加配置」按钮同行左对齐。

```
┌─ 一级 Tab（不动）────────────────────────────────┐
│ 🤖 AI 配置  │ 📦 Tab 2 │ 📦 Tab 3 │                │
└─────────────────────────────────────────────────┘
  ┌ 全部 ┐ ┌ 文字模型 ┐ ┌ 生图模型 ┐  [+ 添加配置]
  ┌ 表格 ───────────────────────────────────────────┐
```

## 改动

1. 删除 h2「AI 配置」+ 描述文字 block
2. 「添加配置」按钮移入二级 pill 同一行，左对齐（pill 后跟按钮，`gap-3`）
3. 二级 tab 行去掉 `border-b` 和 `pb-4`
4. 一级 tab 栏与二级 tab 行间距收窄

## 不动

- 一级 Tab 栏（用户自行添加的顶栏）
- 表格全部结构
- 占位 tab 内容
- 子路由 add/、[id]/
